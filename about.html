About

Namaste, I am Sushant, from Himachal, India

Full stack developer currently working with technologies like Node.JS / Next.JS / TypeScript / React / GraphQL / Google Cloud Platform.

I am a software developer from Kangra, Himachal Pradesh, India. After completing my graduation in 2013, I initially started working with PHP and HTML to develop many websites for our consulting firm. During this time, I worked with PHP frameworks like Joomla, Yii, Codeignitor, and Laravel. During my tenure at this consulting firm, I got the chance to work on a new emerging technology called Node.js. I was maintaining an Angular webapp and a Cordova-based mobile application.

Once I started working with server-side JavaScript technology, I had to often work with various open source projects. This started my OSS contribution journey. I have maintained or am currently maintaining projects like Sequelize, Sequelize CLI, Sequelize Pool, MySQL2, Unexpected Date, and React Samay etc.

Today I am working on JavaScript technologies like Node.JS, Next.JS, TypeScript, React, GraphQL, Bootstrap, and Web Extension Platform etc. to develop many webapps and browser extensions. I also work with other related technologies like Firebase, GCP, Git, Linux, Docker, etc.
